# Controle Financeiro
Uma aplicação Java estruturada para console projetada para o gerenciamento eficiente de finanças pessoais. O sistema permite o cadastro, listagem e remoção de transações financeiras (receitas e despesas), além do cálculo em tempo real do saldo consolidado.

O principal objetivo deste projeto é demonstrar a aplicação prática de padrões de projeto , Programação Orientada a Objetos (POO) avançada e escrita de código limpo (*Clean Code*) em um ambiente Java puro, simulando as exigências técnicas de um ambiente corporativo real.
